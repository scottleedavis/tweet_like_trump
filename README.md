# write_like_trump
tensorflow + ml5.js + trumptwitterarchive.com

A [ml5js](https://ml5js.org/} (built on top of [tensorflow.js](https://js.tensorflow.org/)) application that uses a pretrained model to output texts in the style of Donald Trump, based on the corpus of all his tweets.  

Inspired by [/Currie32/Tweet-Like-Trump](https://github.com/Currie32/Tweet-Like-Trump)

Dataset: Trump Tweets spanning May 4th 2009 to June 9th 2018   via [trumptwitterarchive.com](http://www.trumptwitterarchive.com/archive)
